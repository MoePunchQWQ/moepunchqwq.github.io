MicroPython on ESP8266

Mar 04, 2023

1.搭建烧录环境

(1).安装 Python

到 Python官网 下载对应自己平台的 Python 并安装。本文撰写时 Python 的最新版本为 3.11.2。Windows 系统安装时需要将 Python 添加到环境变量。

(2).下载 ESPTools

我们用 ESPTools 来烧写 MicroPython，ESPTools 支持 Windows/macOS/Linux 三大主流桌面系统，你可以在这里下载到 ESPTools，同时，你也可以在 pip 中安装它
pip install esptool

(3).配置开发板

如果你用的是 Windows，你需要先安装 USB2SerialPort 驱动，ESP8266 模块上常见的有串口芯片有 CP2102 和 CH340/341 两种，可以自行 Google 搜索安装。 将开发板连接到电脑上。打开设备管理器 - 端口（ COM 和 LPT )。如果连接正常，你应该可以看到开发板已经被正确识别，像这样：

双击串口设备，打开端口设置，将位/秒选项更改为115200，像这样：

点击确定，关闭任务管理器窗口。

(4).识别开发板

我们用 ESPTools 检验开发板是否被正确识别，使用这个命令（如果是 Windows 系统，esptools.py 更换为 esptools.exe)： esptool.py flash_id

在这一步，我们可以识别到开发板的具体信息。我们需要读出 Detected flash size 的值。如图，我的是 4MB 。

2.固件的下载与烧录

接下来，我们按照读取出来的 Flash size 下载合适的固件。打开 MicroPython 官网网址，在页面底端按照自己的 Flash size 下载合适的固件。 我下载的是 2MiB+ 里面 release 渠道的 esp8266-20220618-v1.19.1.bin 现在，我们需要将开发板中的固件擦除，在终端中执行以下命令：
Linux 用这个：
esptool.py --port /dev/ttyUSB0 erase_flash
Windows 用这个：
esptool.exe --port 这里填你的 COM 号 erase_flash
我的执行结果如下：

现在我们来烧录固件。使用以下命令：
Linux 用这个：
esptool.py --port /dev/ttyUSB0 --baud 115200 write_flash --flash_size=detect 0 这里填你下载的 bin
Windows 用这个：
esptool.exe --port 这里填你的 COM 号 --baud 115200 write_flash --flash_size=detect 0 这里填你下载的 bin
